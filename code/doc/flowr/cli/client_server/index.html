message_queue implementation of the communications between the runtime client, debug client and the runtime server and debug server.

Structs

ClientConnection
ClientConnection stores information related to the connection from a runtime client to the runtime server and is used each time a message is to be sent or received.
ServerConnection
ServerConnection store information about the server side of the client/server communications between a runtime client and a runtime server and is used each time a message needs to be sent or received.
